# Lumina Note 文档编辑/排版界面 视觉与交互升级方案

## 设计目标
- 高级但安静：弱存在感 UI，不抢内容注意力。
- 克制：尽量减少元素，新增元素均说明必要性。
- 纸面心智=屏幕心智：编辑即纸面视图，不做编辑/预览切换。
- 尺寸统一：页面尺寸/边距/行距/段前后统一使用 mm/pt，界面显示同单位。
- 滚动即分页视觉、缩放不改变逻辑尺寸。

---

## 结构总览（单屏布局）
- 顶部：轻量规则化工具栏（保存/导出/打印为主操作）。
- 左侧：可折叠文件树（层级清晰，默认窄）。
- 中央：纸面视图（分页、页眉页脚、对齐、行距如打印预览）。
- 右侧：可选窄栏“样式/段落面板”（默认隐藏）。
- 底部状态：分页/缩放/字数/同步状态（低干扰）。

新增元素必要性说明：
- 右侧样式面板：仅在需要精确排版时出现，减少正文干扰。
- 底部状态条：提供“纸面心智”确认信息（页码/缩放/尺寸），避免额外弹窗。

## 视觉系统

### 颜色变量（建议）
- --bg-main: #F6F4F1 (主背景，暖灰/米白)
- --bg-paper: #FBFAF8 (纸面背景，比主背景略亮)
- --text-primary: #1F2328 (正文/主文字)
- --text-secondary: #5B6168 (次级说明)
- --stroke-subtle: #E6E2DD (分割线/描边)
- --accent: #0E3A2F (深墨绿，低饱和)
- --accent-alt: #1E2F4F (海军蓝，低饱和)
- --shadow-soft: rgba(31,35,40,0.06) (纸面轻投影)

使用原则：
- 强调色仅用于主按钮、选中态、焦点提示；面积<5%。
- 分割线使用极浅灰，避免重边界。

### 字体与字号层级
UI 字体：IBM Plex Sans / Source Han Sans SC
正文内容字体：由文档设置控制，但与 UI 清晰区分。

字号层级（pt）：
- UI 标题：13pt
- UI 正文：11pt
- UI 辅助：10pt
- 编辑器正文基准：11pt / 12pt / 14pt（随文档设置）

### 间距尺度（pt / mm）
- 4pt, 8pt, 12pt, 16pt, 24pt, 32pt
- 2mm, 4mm, 6mm, 10mm

规则：
- UI 间距使用 pt；纸面排版与文档尺寸使用 mm。

### 栅格规则
- 主界面使用 12 列栅格，内容区居中对齐。
- 中心纸面视图宽度按页面尺寸 (A4/Letter 等) + 左右 12-16mm 可视边距。
- 侧栏默认宽度：左 56mm，右 48mm。

---

## 页面视图（纸面心智）

### 纸面区域
- 默认进入分页视图，纸张居中。
- 背景为暖灰层次，纸面为略亮米白。
- 纸面轻微阴影（仅 1-2px 模糊，透明度极低）。
- 页间距 10-14mm，确保“翻页”节奏。

### 页眉/页脚
- 结构元素，不可编辑。
- 显示为浅灰线标示区域，并用淡淡标识“Header / Footer”。
- 悬停时显示提示，不出现可输入光标。

### 正文区域
- 可编辑，光标与选区在纸面内。
- 选区高亮使用淡灰蓝或淡墨绿（低饱和）。

---

## 主要组件设计

### 顶部工具栏
- 高度：36-40pt
- 背景与主背景一致，底部 1px 极浅灰分割。
- 左侧：文档标题 + 状态（自动保存/已保存）。
- 中央：常用排版操作（字体、字号、行距、对齐）。
- 右侧：主操作按钮（保存、导出、打印）。

按钮风格：
- 主按钮：低饱和墨绿填充，轻微圆角 4pt。
- 次按钮：仅描边或淡灰底。

### 左侧文件树
- 默认窄栏，支持一键折叠。
- 文件层级缩进 8pt，文件名 11pt。
- 高亮当前文档：淡灰底 + 左侧 2pt 强调色线。

### 右侧样式/段落面板
- 默认隐藏，仅在用户点击“样式/段落”时出现。
- 面板宽度 48mm，分组清晰（字体、段落、样式）。
- 分组标题使用 10pt，全大写或细分隔线。

### 底部状态提示
- 高度 24pt
- 左侧：页码信息（第 X 页 / 共 Y 页）
- 中央：缩放百分比（100% 等）
- 右侧：文档尺寸（A4 210x297mm）与边距信息

---

## 关键交互说明

### 翻页
- 滚动即分页视觉。
- 页与页之间有固定间距，不使用连续长画布。
- 滚动时顶部状态条显示当前页码。

### 缩放
- 缩放仅改变屏幕显示比例，不改变逻辑尺寸。
- 缩放控件在底部状态条，支持 50%/75%/100%/125%/150% 快捷。

### 导出
- 点击“导出”按钮后弹出轻量浮层（非全屏）。
- 默认显示：PDF / DOCX / 打印预览。

### 编辑焦点
- 文本光标在正文区域内高亮。
- 页眉/页脚不可编辑，点击仅提示。

---

## 可编辑与不可编辑区分

不可编辑结构元素：
- 页眉/页脚
- 页面边距标识线
- 页码区域

可编辑内容：
- 正文文本与内嵌元素（表格、图片、引用）
- 正文区域内的段落样式

---

## 节奏与质感细节
- 所有边界线使用极浅灰，避免强对比。
- 页面阴影仅为“纸张浮起”感觉，透明度极低。
- 交互反馈（按钮、选区）仅 120-160ms 微动画。

---

## 交付总结
- 该方案以“纸面心智”为中心，不存在编辑/预览切换。
- UI 极简、克制，主操作清晰，内容为核心。
- 以网格、字号层级、留白与材质感建立品质。
